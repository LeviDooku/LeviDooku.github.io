Como está hecha la página

Esta página se construye por medio de dos lenguajes de programación q van uno de la mano al otro. Estos son HTML y CSS.

Podríamos decir qué con el lenguaje HTML se construye el esqueleto de la página mientras que con CSS le damos el estilo que queramos. HTML funciona por medio de etiquetas donde podemos crear clases en etiquetas para dar estilo a las mismas o al grupo de etiquetas que incluya la "padre".Por este motivo, cada archivo HTML tiene asociado su respectivo archivo de estilos CSS.
